How it Works

To achieve real time styling, Stylable makes use of the content editable attribute in an internal stylesheet. By setting this attribute equal to "true" in the style tag, players are able to dynamically type new styles that update the Document Object Model (DOM). Given that an internal stylesheet has the ability to overwrite defined styles in an external sheet, this method allows for players to easily style accordingly without any hierarchy issues.
